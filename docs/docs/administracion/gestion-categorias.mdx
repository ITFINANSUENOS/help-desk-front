# Gestión de Categorías

Las **Categorías** clasifican los tickets y definen a qué Empresa y Departamento pertenecen.

## Acceso

Para acceder, seleccione **Administración > Categorías** en el menú lateral.

## Crear Categoría

1.  Haga clic en **Nueva Categoría**.
2.  **Nombre**: Ingrese un nombre descriptivo (ej: "Soporte Técnico").
3.  **Empresas**: Seleccione una o varias empresas que tendrán acceso a esta categoría.
4.  **Departamentos**: Seleccione uno o varios departamentos responsables de esta categoría.
5.  **Estado**: Activo/Inactivo.
6.  Haga clic en **Guardar**.

## Relaciones

Las categorías actúan como "padres" de las **Subcategorías**. Es fundamental configurarlas correctamente para que el flujo de tickets funcione.
