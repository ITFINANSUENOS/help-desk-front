# Gestión de Cargos

El módulo de **Cargos** define las posiciones laborales disponibles en la organización.

## Acceso

Para acceder, seleccione **Administración > Cargos** en el menú lateral.

## Listado de Cargos

- **Nombre**: Título del cargo.
- **Estado**: Activo/Inactivo.

## Crear Cargo

1.  Haga clic en **Nuevo Cargo**.
2.  Ingrese el **Nombre** del cargo.
3.  El estado será **Activo** por defecto.
4.  Guarde.

:::tip Nota
Los cargos creados aquí son los que estarán disponibles para ser utilizados en el **Organigrama** y en la asignación de flujos de trabajo.
:::
