# Gestión de Departamentos

El módulo de **Departamentos** administra las áreas funcionales de la empresa (ej: Sistemas, Contabilidad).

## Acceso

Para acceder, seleccione **Administración > Departamentos** en el menú lateral.

## Listado de Departamentos

- **Nombre**: Identificador del departamento.
- **Estado**: Activo/Inactivo.

## Crear/Editar

1.  Haga clic en **Nuevo Departamento**.
2.  Ingrese el **Nombre** del departamento.
3.  Establezca el **Estado**.
4.  Guarde el registro.

## Importante

Los departamentos se utilizan para clasificar Tickets y asignar responsabilidades. Desactivar un departamento puede afectar la creación de nuevos tickets asociados a él.
