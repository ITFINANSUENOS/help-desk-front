# Preguntas Frecuentes y Solución de Problemas

## Tickets

### ¿Por qué no puedo ver un ticket que creé?

**Posibles causas:**
1. **Filtros activos**: Verifica que no tengas filtros aplicados que excluyan el ticket.
2. **Vista incorrecta**: Asegúrate de estar en la vista "Creados por mí" o "Todos los Tickets".
3. **Búsqueda activa**: Limpia el campo de búsqueda.

**Solución:**
- Haz clic en "Limpiar Filtros" en el panel de Filtros Avanzados
- Cambia a la vista "Creados por mí"

### ¿Por qué no aparece el botón "Reabrir Ticket"?

**Posibles causas:**
1. El ticket no está cerrado
2. No tienes permisos para reabrir tickets
3. El ticket ya fue reabierto previamente

**Solución:**
- Verifica que el estado del ticket sea "Cerrado"
- Contacta a tu administrador si necesitas permisos

### No encuentro las subcategorías que necesito

**Posibles causas:**
1. No has seleccionado una empresa primero
2. El filtro "Solo mis procesos" está activo
3. No tienes permisos para esas subcategorías

**Solución:**
- Selecciona primero una empresa en el filtro
- Desactiva "Solo mis procesos" si quieres ver todas las subcategorías
- Verifica tus permisos con el administrador

## Notificaciones

### No recibo notificaciones de tickets

**Posibles causas:**
1. No estás asignado ni eres observador del ticket
2. Las notificaciones fueron marcadas como leídas automáticamente
3. Problema de conexión con el servidor

**Solución:**
- Verifica que estés asignado o seas observador en el ticket
- Revisa la página completa de notificaciones (no solo el panel)
- Recarga la página

### Las notificaciones desaparecen del panel

**Esto es normal**: El panel de la campana solo muestra notificaciones **no leídas**. Cuando haces clic en una notificación, se marca como leída y desaparece del panel.

**Para ver el historial completo:**
- Ve a la página de Notificaciones desde el menú
- Selecciona la pestaña "Todas"

### ¿Cómo elimino notificaciones antiguas?

**No se pueden eliminar**, pero puedes:
- Marcarlas como leídas para ocultarlas del panel
- Usar "Marcar todas como leídas" para limpiar el panel completo

## Filtros

### El filtro "Solo mis procesos" no muestra nada

**Posibles causas:**
1. No has participado en ningún ticket aún
2. No has sido creador ni asignado de tickets

**Solución:**
- Desactiva el filtro para ver todas las subcategorías disponibles
- Crea o trabaja en algunos tickets primero

### Los filtros de usuario no cargan opciones

**Posibles causas:**
1. No has escrito al menos 2 caracteres en la búsqueda
2. No hay usuarios que coincidan con tu búsqueda
3. Problema de conexión

**Solución:**
- Escribe al menos 2 caracteres para activar la búsqueda
- Verifica la ortografía del nombre
- Recarga la página

### No puedo seleccionar empresa y "Solo mis procesos" al mismo tiempo

**Esto es por diseño**: Estos filtros son mutuamente excluyentes:
- **Empresa**: Muestra subcategorías de una empresa específica
- **Solo mis procesos**: Muestra subcategorías donde has participado (sin importar la empresa)

## Vistas de Tickets

### ¿Qué diferencia hay entre "Errores que reporté" y "Errores asignados a mí"?

- **Errores que reporté**: Tickets donde **tú** reportaste un error de proceso
- **Errores asignados a mí**: Tickets con errores que **te asignaron** para resolver

### ¿Por qué un ticket desapareció de "Asignados a mí"?

**Posibles causas:**
1. Completaste tu tarea paralela (si era un ticket paralelo)
2. El ticket fue reasignado a otra persona
3. El ticket fue cerrado

**Esto es normal** para tickets paralelos: Una vez completas tu parte, el ticket se oculta de tu vista para evitar desorden.

## Workflows

### No puedo avanzar un ticket al siguiente paso

**Posibles causas:**
1. Faltan campos obligatorios
2. No tienes permisos para ese paso
3. El paso requiere aprobación de otro usuario
4. Hay un error de proceso pendiente

**Solución:**
- Completa todos los campos marcados como obligatorios
- Verifica que tengas permisos para el siguiente paso
- Revisa si hay errores reportados en el ticket

### ¿Qué significa "Error de proceso"?

Un error de proceso indica que:
- Algo salió mal durante el flujo del ticket
- Se requiere intervención manual
- El ticket fue reabierto

**Solución:**
- Revisa el historial del ticket para ver el detalle del error
- Contacta al usuario asignado para resolver el error

## Rendimiento

### La aplicación está lenta

**Soluciones:**
1. Limpia los filtros activos (especialmente búsqueda en mensajes)
2. Reduce el rango de fechas en los filtros
3. Cierra pestañas del navegador que no uses
4. Recarga la página (F5)

### El filtro "Solo mis procesos" tarda mucho

Si la primera carga es lenta, es normal. El sistema está buscando en todo tu historial de tickets. Las siguientes cargas serán más rápidas.

## Errores Comunes

### "No tienes permisos para ver este ticket"

**Causa**: No estás asignado, no eres creador, ni observador del ticket.

**Solución**: Contacta al administrador si crees que deberías tener acceso.

### "Error al cargar datos"

**Soluciones:**
1. Verifica tu conexión a internet
2. Recarga la página (F5)
3. Limpia la caché del navegador
4. Contacta a soporte si persiste

## ¿Necesitas más ayuda?

Si tu problema no está listado aquí:
1. Contacta a tu administrador del sistema
2. Revisa las guías específicas en este manual
3. Reporta el problema a soporte técnico con capturas de pantalla
